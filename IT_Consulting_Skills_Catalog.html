IT Consulting Skills Catalog

Competency inventory across presales, technical, and delivery domains

This is a competency inventory of individual "skills" — discrete areas of expertise — organized into 12 domain sheets, from Presales/GTM/Commercial to AI/ML Infrastructure, Zero Trust, Firewall, and more. Each row is one named skill that could be matched against a project, RFP, or staffing need.

ID

A unique identifier for the skill. Most are numeric sequence IDs; some (prefixed P-) mark later-added strategic/executive skills.

Competency Area

The name of the specific skill or capability, e.g. "SD-WAN Architecture & Design" or "AI Model Explainability & Interpretability Tooling".

Description

A plain-language, one-line explanation of what the skill actually involves in practice.

Kind

Currently all entries are tagged Unique, meaning each skill is a distinct, non-overlapping competency in the catalog.

Matched Domain(s)

The broader practice area(s) the skill belongs to. A skill can map to more than one domain — shown as separate tags — when it spans disciplines (e.g. both Zero Trust and Compliance).
